K.N.S. AddressBook v1.4.0

K.N.S. AddressBook is a free Java desktop application available on Windows and Mac OS X operating systems. It is designed for freelancers who are bound to their desks, making many calls to clients with their work phone.

Oftentimes, freelancers have one hand on the phone, and their other hand can’t possibly use both the keyboard and mouse at the same time! Thus, most of the interactions with the application are done through an in-built Command Line Interface (CLI), which allows you do add, edit and remove contacts quickly with only the use of a keyboard.

Even if you aren’t a freelancer, you’re always welcome to try out our application!
